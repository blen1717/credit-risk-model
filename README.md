# Credit Risk Model for Xente eCommerce Platform

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Project Overview
This project builds a credit risk scoring model using transaction data from the Xente platform (Kaggle challenge).  
Because no historical default label exists, we create a proxy target using RFM (Recency, Frequency, Monetary) analysis and K‑means clustering. The final output is a risk probability and a credit score.
